## Short answer

The source-of-truth matrix names the winning record for each kind of Regent fact. It keeps public pages, product views, agents, and operators from treating mirrors as authority.

## Canonical definition

The source-of-truth matrix is the Regent rulebook for which product or state source wins when records describe the same thing.

## Why it matters

Agents need stable rules. When local state, product state, and chain state disagree, the system needs a declared winner.

## How it works in Regents

Platform owns guided human setup and public company records. Techtree owns research workflow. Autolaunch owns launch workflow. Shared SIWA owns identity proof. Onchain state wins for money state. Regents CLI local state is downstream.

## What this does not claim

The matrix does not make every mirror instantly current. It defines which record wins when accuracy matters.
